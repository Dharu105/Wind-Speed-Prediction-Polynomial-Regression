# Wind Speed Prediction using Polynomial Regression

## Project Overview
This project predicts 100m wind speed using 50m wind speed data using Polynomial Regression.

## Dataset
- Wind speed measurements at different heights
- Cleaned and processed before training

## Steps Performed
1. Correlation Matrix Analysis
2. Feature Selection
3. Train-Test Split (70:30)
4. Polynomial Feature Transformation
5. Model Training
6. Prediction
7. Model Evaluation

## Evaluation Metrics
- MAE (Mean Absolute Error)
- MSE (Mean Squared Error)
- R² Score

## Libraries Used
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Output
- Actual vs Predicted Line Chart
- R² Score ≈ 0.88
- MSE ≈ 1.02
